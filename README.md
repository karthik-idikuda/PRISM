# PRISM

Phase-Resolved Intelligence for Sustainable Manufacturing. AI platform with DWT spectral analysis and LightGBM optimization for pharmaceutical batch process monitoring, developed for AVEVA x IIT Hyderabad.

## Features

- Discrete Wavelet Transform for spectral feature extraction
- LightGBM-based batch quality prediction
- Real-time process parameter monitoring
- Anomaly detection for out-of-spec batches
- Interactive Streamlit dashboard

## Tech Stack

Python, LightGBM, DWT, Streamlit, Pandas, NumPy

## Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

```bash
git clone https://github.com/karthik-idikuda/PRISM.git
cd PRISM
pip install -r requirements.txt
```

### Usage

```bash
streamlit run app.py
```
